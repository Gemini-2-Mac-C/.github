## 🔍 Overview

<a href="#gemini2">Gemini 2</a> is a smart duplicate file finder for Mac that identifies and removes redundant files from the drive — recovering wasted storage space by finding exact duplicates and similar files that accumulate over time through downloads, photo imports, backup operations, and file copying across locations.

The <a href="#gemini2">smart scanning algorithm</a> analyzes file content rather than just filename and size to identify true duplicates — comparing file data to confirm that two files with different names or locations are genuinely identical, and using perceptual similarity algorithms to identify photos that are nearly identical but not byte-perfect copies. This content-based detection catches duplicates that simple filename matching would miss: the same photo downloaded twice with different filenames, the same document saved to two locations with slight naming differences, or the same music track in two library locations. The <a href="#gemini2">similar image detection</a> extends beyond exact duplicates to identify photos that are visually nearly identical — burst shots where multiple frames capture nearly the same moment, slightly different crops or edits of the same original, or the same photo exported at different resolutions or quality settings. These near-duplicates are harder to find manually but often represent significant photo library bloat when hundreds or thousands of similar burst shots accumulate.

The <a href="#gemini2">review before removal workflow</a> presents found duplicates for user review before any deletion occurs — showing the files side by side, indicating which copy appears to be the better quality or more recently modified version, and letting the user confirm the selection before removal. No automatic deletion without review: Gemini 2's smart selection suggests which copy to keep but the user controls every deletion decision. The <a href="#gemini2">safe removal to Trash</a> moves removed files to the Mac Trash rather than permanently deleting them — files remain recoverable from the Trash for a final safety net if the wrong file was removed.</p>

<a href="#gemini2">Music duplicate detection</a> identifies duplicate tracks in music libraries — the same song in multiple formats, the same album downloaded twice, or tracks that appear in both the main library and a backup location. <a href="#gemini2">External drive scanning</a> extends the duplicate search to connected external drives and USB storage — finding duplicates that span the internal drive and external storage where backup copies and transferred files accumulate.

<a href="#gemini2">Storage space visualization</a> shows how much space duplicates occupy before and after cleanup — communicating the storage recovery in concrete gigabyte terms that motivate the cleanup effort. <a href="#gemini2">Folder comparison mode</a> compares two specific folders for duplicates — useful for comparing a working folder against a backup folder or comparing two photo library locations to identify overlap before merging.</p>

## 📌 FAQ

<details>
<summary>Does Gemini 2 permanently delete files?</summary>

No. Gemini 2 moves removed files to the Mac Trash, where they remain recoverable until the Trash is emptied.

</details>

<details>
<summary>Can Gemini 2 find similar photos, not just exact duplicates?</summary>

Yes. Perceptual similarity detection identifies photos that are visually nearly identical — burst shots, different-resolution exports, lightly edited versions.

</details>

<details>
<summary>Does Gemini 2 review files before removing them?</summary>

Yes. Gemini 2 presents found duplicates for user review before any removal occurs. The user confirms every deletion decision.

</details>

<details>
<summary>Can Gemini 2 scan external drives?</summary>

Yes. External drives and USB storage scan for duplicates alongside the internal drive.

</details>

<details>
<summary>Does Gemini 2 find music duplicates?</summary>

Yes. Music duplicate detection identifies the same tracks in multiple formats or library locations.
